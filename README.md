# WP Image Guard

Protect your WordPress images from easy copying.

WP Image Guard adds a protected viewing layer (lightbox) with watermarking and interaction restrictions to make image copying harder, slower, and less appealing.

---

## 🔒 Features

- Watermark overlay (text and/or logo)
- Multi-line watermark support
- Adjustable:
  - font size
  - color
  - opacity
  - rotation angle
- Dynamic watermark variables:
  - `{site}`, `{date}`, `{time}`, `{user}`, `{ip}`, `{image_id}`, `{filename}`
- Right-click disable (best-effort)
- Drag & drop prevention
- Works automatically with:
  - posts and pages
  - WordPress galleries
  - WooCommerce product images
  - page builders (Elementor, Divi, etc.)
- Per-image protection settings in Media Library
- Language support: Finnish / English

---

## ⚙️ Installation

1. Download the plugin ZIP
2. Go to **WordPress Admin → Plugins → Add New**
3. Click **Upload Plugin**
4. Upload the ZIP file
5. Activate the plugin

---

## 🚀 Quick Start

1. Go to **Settings → WP Image Guard**
2. Choose what to protect:
   - all images
   - galleries only
   - specific CSS class
3. Customize watermark settings
4. Save changes

Done! 🎉

---

## 🖼️ How It Works

- Images display normally on the page
- When clicked:
  - a protected overlay opens
  - watermark is applied
  - copying actions are restricted

---

## 🛠️ Settings Overview

### Watermark
- Text (supports dynamic variables)
- Font size
- Color
- Opacity
- Rotation
- Number of rows

### Logo
- Upload a PNG watermark

### Protection Scope
- All images
- Galleries only
- CSS class-based protection
- WooCommerce images

---

## ⚠️ Limitations (Important)

WP Image Guard improves protection, but does **not** provide complete security.

### What it helps prevent:
- Easy right-click saving
- Quick copying by non-technical users
- Clean reuse of images (due to watermark)

### What it cannot prevent:
- Screenshots
- Advanced users extracting images via browser tools
- 100% copying prevention

👉 If an image is visible in a browser, it has already been delivered to the user.

---

## 🎯 Use Cases

Best suited for:
- blogs
- portfolios
- product images
- general content protection

Not intended for:
- DRM-level protection
- highly sensitive media

---

## 💡 Tips

- Use subtle watermark opacity (~30–50%)
- Include your site name
- Use dynamic variables for better traceability

# 🇫🇮 Suomeksi

## 🔒 Mikä tämä on?

WP Image Guard suojaa WordPress-sivustosi kuvia estämällä helpon kopioinnin ja lisäämällä vesileiman suurennettuihin kuviin.

---

## ⚙️ Nopea aloitus

1. Mene kohtaan **Asetukset → WP Image Guard**
2. Valitse mitä kuvia suojataan
3. Muokkaa vesileima-asetuksia
4. Tallenna

Valmista! 🎉

---

## 🖼️ Miten se toimii

- Kuvat näkyvät normaalisti sivulla
- Klikattaessa:
  - avautuu suojattu näkymä
  - vesileima lisätään
  - kopiointi vaikeutuu

---

## ⚠️ Rajoitukset (tärkeää)

Lisäosa ei tee kuvista täysin suojattuja.

### Estää:
- helpon oikea klikkaus → tallenna
- nopean kopioinnin
- kuvien puhtaan uudelleenkäytön

### Ei estä:
- kuvakaappauksia
- edistyneitä käyttäjiä
- täydellistä kopiointia

👉 Jos kuva näkyy selaimessa, se on jo ladattu käyttäjän laitteelle.

---

- verkkokauppoihin

Ei tarkoitettu:
